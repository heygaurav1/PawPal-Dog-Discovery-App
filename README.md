# PawPal-Dog-Discovery-App 🐶✨

A delightful iOS app that brings you a new cute dog photo every time you tap! Built with **SwiftUI**, **SwiftData**, and modern architecture.

![App Preview](screenshots/home.png)

## ✨ Features

- **Instant Dog Discovery** — Fetch random adorable dog photos with one tap using the Dog CEO API
- **Favorites System** — Save your favorite doggos permanently using **SwiftData**
- **Beautiful Image Gallery** — Browse all saved dogs in a clean grid layout
- **Smooth Animations** — Delightful loading states and transitions
- **Dark Mode Support** — Looks great in light and dark themes
- **Share Your Doggo** — Easily share cute dogs with friends
- **Offline Favorites** — All saved dogs available even without internet

## 🛠 Tech Stack

- **SwiftUI** — Modern declarative UI
- **SwiftData** — Persistent local storage
- **AsyncImage + URLSession** — Efficient image loading
- **MVVM + Repository Pattern** — Clean & scalable architecture
- **async/await** — Modern concurrency
- **Dependency Injection** — Better testability
